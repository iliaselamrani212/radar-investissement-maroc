# 🎯 Radar des Projets d'Investissement - Maroc

Projet pour SDG Capital - Système intelligent de détection et 
priorisation des opportunités d'investissement au Maroc.

## 🏗️ Structure

- `scraping/` - Collecte de données depuis sources officielles
- `ai_extraction/` - Extraction structurée via LLM
- `backend/` - API FastAPI
- `frontend/` - Dashboard Next.js
- `map_pitch/` - Cartographie et présentation

## 🚀 Démarrage

```bash
docker-compose up -d
```

## 👥 Équipe

- Membre 1 - Scraping
- Membre 2 - IA & Extraction  
- Membre 3 - Backend
- Membre 4 - Frontend
- Membre 5 - Carte & Pitch
